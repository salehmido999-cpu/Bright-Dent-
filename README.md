# نظام إدارة عيادة الأسنان - Bright Dent

نظام ويب متكامل لإدارة عيادة الأسنان باستخدام React و Node.js و MySQL.

## المميزات الرئيسية

- ✅ تسجيل المرضى الجدد
- ✅ نموذج التاريخ المرضي مع تنبيهات أمان
- ✅ إدارة الزيارات والإجراءات
- ✅ خريطة أسنان تفاعلية (Odontogram)
- ✅ نظام الفواتير الآلي
- ✅ واجهة متجاوبة بالعربية

## المتطلبات

- Node.js >= 16
- MySQL >= 5.7
- npm أو yarn

## البدء السريع

```bash
# تثبيت المتطلبات
npm install

# تشغيل الخادم
npm run server

# تشغيل التطبيق
npm start
```

## الهيكل

```
├── client/          # تطبيق React
├── server/          # خادم Node.js
└── database/        # ملفات قاعدة البيانات
```
